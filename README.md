# India Price Tracker 🔍

Live price comparison across **Amazon India**, **Flipkart**, and **Croma** — powered by Claude AI + real-time web search.

🌐 **[Live Demo →](https://india-price-tracker.vercel.app)**

## Features
- 🔍 Search **any device** — phones, laptops, headphones, appliances
- ⚡ Live prices fetched via Anthropic Claude AI + web search
- 📊 Price trend chart + all-time low estimates  
- 💡 Smart buy signals (sale seasons, bank offers, exchange bonus)
- 🏆 Automatic best-deal recommendation

## Usage
1. Open the live URL
2. Paste your [Anthropic API key](https://console.anthropic.com)
3. Search any device and compare instantly

## Tech Stack
- Pure HTML/CSS/JS — no framework, no build step
- Claude AI (`claude-sonnet-4-20250514`) with `web_search_20250305`
- Chart.js for price trend visualization
- Deployed on Vercel
